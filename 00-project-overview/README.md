Project Overview
================

Goal
----
Build a portfolio of AI responsibility practices with clear artifacts, evaluation, and reflection.

Deliverables
------------
- A single-page project charter (scope, users, risks).
- A dataset policy (sources, consent, retention).
- A model registry entry template (name, version, owner).
- A lightweight risk register (top 5 risks, mitigations).

Tech stack (suggested)
----------------------
- Python for experiments
- Jupyter for notes
- Markdown for documentation

Start here checklist
--------------------
- Define a simple use case and target users.
- Decide the model type (e.g., classifier, text model).
- Create a synthetic dataset or use a public dataset.
