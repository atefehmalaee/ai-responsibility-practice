User Impact & Stakeholder Engagement
====================================

Goal
----
Understand and reduce negative user impact.

What to build (professional, portfolio-ready)
---------------------------------------------
- User research plan and survey
- Impact assessment table
- Accessibility/usability checklist

Techniques
----------
- User research and feedback collection
- Accessibility and usability checks
- Impact assessment (benefits and harms)

Exercises
---------
- Draft a short user survey for your use case.
- Identify top 3 possible harms and mitigations.

Quick start
-----------
1. Install deps:
   - `python -m venv .venv`
   - `source .venv/bin/activate`
   - `pip install -r requirements.txt`
2. Run the user impact audit:
   - `python user_impact_audit.py --out reports`

Outputs
-------
- `reports/user_survey.md`
- `reports/impact_assessment.csv`
- `reports/accessibility_checklist.md`

Deliverables
------------
- User feedback plan
- Impact assessment summary
