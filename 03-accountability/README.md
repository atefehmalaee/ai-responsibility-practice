Accountability & Governance
===========================

Goal
----
Define ownership, review gates, and decision authority.

Techniques
----------
- RACI chart for model lifecycle
- Approval checkpoints (pre-launch, post-launch)
- Audit trail for changes and approvals

Exercises
---------
- Create a RACI for data, model, and release steps.
- Define a minimum approval checklist for launch.

Deliverables
------------
- RACI table
- Approval checklist template
